# Run
## Run No 1
Install ```light-server```:
``` [bash]
npm install -g http-server
```

Run in port 3000 :
``` [bash]
cd ./No1-frontend
light-server -s . -p 3000
```

## Run No 2
Run with docker by:

```[bash]
cd ./No2-backend
docker compose up
```
